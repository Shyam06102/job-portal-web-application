# MERN job-portal-web-application

A simple Job Portal Web Application built using the MERN Stack.

Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication
- Axios

Features
- User Registration and Login
- Job Posting
- Job Listing
- Job Deletion
- REST API integration

Run Backend
cd backend
npm install
node server.js

Run Frontend
cd frontend
npm install
npm start
